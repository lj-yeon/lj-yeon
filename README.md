## 💼 Experience

### 퍼널스 | 웹 개발자 (2023.12 ~ 현재)

2인 개발 체제로 시작해 현재 **1인 풀스택 체제**로 전환,
기획·디자인·프론트엔드·백엔드·인프라·배포·운영까지 전 과정을 단독으로 수행합니다.

**주요 프로젝트** (총 8건)

| # | 프로젝트 | 설명 | 핵심 기술 |
|---|---------|------|----------|
| 1 | 마케터 지원 도구 | 대시보드와 콘텐츠 저장소를 통합한 마케팅 운영 플랫폼 | React, Python, PostgreSQL |
| 2 | 마케팅 성과 대시보드 | ROAS 및 광고 성과를 실시간 시각화하는 BI 대시보드 | React, Node.js, PostgreSQL |
| 3 | 랜딩페이지 발급 솔루션 | 마케터가 직접 제작·배포하는 셀프 빌더 | React, Node.js, MySQL |
| 4 | SNS 릴스 자동생성 앱 | AI 기반 숏폼 콘텐츠 자동 생성·게시 자동화 | React, Python, AI API |
| 5 | 리드 데이터 관리 솔루션 | 랜딩페이지 리드와 TM 시스템을 연동한 통합 관리 | React, Node.js, MySQL |
| 6 | 퍼널스 공식 홈페이지 | 기업 브랜드 사이트 기획·디자인·개발·배포 | React |
| 7 | 덴토탈 (B2B SaaS) | 병의원 기공물 주문·제작 관리 시스템 | React, Node.js, MySQL |
| 8 | 운행일지 관리 프로그램 | 차량 운행 기록 자동화 관리 도구 | React, Node.js |

### 에코베이스 | 디자인 · 마케팅 (2년 8개월)

B2B 마케팅 기획, 인쇄물·디지털 디자인, 영상 촬영·편집, 유튜브 채널 운영을 담당하며
**사용자 관점과 비즈니스 감각**을 다졌습니다. 이 경험은 현재 개발 업무에서
"왜 이 기능이 필요한가"를 제품 관점으로 판단하는 기반이 되고 있습니다.
